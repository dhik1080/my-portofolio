KONSEP DASAR CODDING. 1.variabel.ini adalah nilai,angka,character string maupun memory addres yang memungkinkan programmer untuk membuat sesuatu program. 2.control structure,yakni serangkain blok pemrograman yang menganalisis variabel dan menentukan arah selanjutnya. 3.struktur data, ini merupakan format khusus untuk menganalisasi dan menyimpan data sesuai tujuan tertentu. 4.syntax, yaitu code atau script yang dipahami oleh komputer guna mencapai tujuan programmer. 5.tools, atau serangkain program komputer yang diperlukan untuk menciptakan, memelihara, mendukung,,serta menjalankan debug terhadap program maupun aplikasi lainya. selain yang disebutkan diatas,ada beberapa pakar yang menyebutkan tentang konsep dasar pemrograman yang terdiri dari input,proses,dan output.
